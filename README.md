ship_it
==============

Geddy-based shipments page

## Quick start

To run the app (run from main directory):

* > geddy

To deploy:

* > jitsu deploy

## Helpful hints

This app was built using Geddy MVC (on top of Node.js), AngularJS and Bootstrap.
